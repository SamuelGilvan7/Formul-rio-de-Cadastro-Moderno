# 📋 Formulário de Cadastro Moderno

Um formulário simples, bonito e seguro para cadastro de usuários com **validação em PHP**.

## ✨ Funcionalidades

- Design moderno e responsivo
- Validação completa no servidor (PHP)
- Proteção contra XSS com `htmlspecialchars()`
- Sanitização de dados de entrada
- Mensagens de erro e sucesso claras
- Campos: **Nome**, **Email** e **Idade**
- Valores mantidos no formulário após erro (melhor experiência do usuário)

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** (com gradiente e efeitos suaves)
- **PHP 8+** (filter_input, validações nativas)
- Google Fonts (Inter)
